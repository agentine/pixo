# pixo

Pure Go image processing library. Drop-in replacement for [nfnt/resize](https://github.com/nfnt/resize) and [disintegration/imaging](https://github.com/disintegration/imaging).

## Features

- Resize, Fit, Fill, Thumbnail with multiple interpolation filters
- Rotate, Flip, Transpose, Transverse
- Crop with anchor-based positioning
- Brightness, Contrast, Gamma, Saturation, Hue adjustments
- Blur, Sharpen, Grayscale, Invert
- Image compositing (Paste, Overlay)
- I/O with format detection and EXIF auto-orientation
- Compatibility packages for painless migration

## Install

```
go get github.com/agentine/pixo
```
